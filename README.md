# 🚀 معرض أعمالي | Portfolio - أحمد الشريف

أهلاً بك في مستودع ملفات موقعي الشخصي. أنا مبرمج شغوف ببناء حلول برمجية متكاملة، متخصص في تطوير تطبيقات الموبايل باستخدام **Flutter** ومواقع الويب المتطورة.

## 🛠 التقنيات المستخدمة

تم بناء هذا الموقع باستخدام تقنيات الويب الأساسية مع التركيز على التصميم العصري والأداء:

- **HTML5 & CSS3**: للهيكل والتصميم المتجاوب (Responsive Design).
- **JavaScript**: لإضافة الحركية وإدارة عرض المشاريع ديناميكياً.
- **Font Awesome**: للأيقونات التفاعلية.
- **Google Fonts**: خط (Cairo) لتحسين تجربة القراءة بالعربية.

## 🌟 مميزات المشروع

- **نظام عرض مشاريع ديناميكي**: يتم إضافة المشاريع الجديدة عبر ملف JavaScript بسهولة.
- **تصميم متجاوب**: الموقع يعمل بكفاءة عالية على جميع الشاشات (موبايل، تابلت، كمبيوتر).
- **تجربة مستخدم (UX) مميزة**: ألوان مريحة للعين وتصفح سلس.

## 📁 هيكل المشروع

- `index.html`: الملف الأساسي للموقع.
- `style.css`: يحتوي على جميع التنسيقات والـ Media Queries.
- `script.js`: المسؤول عن تحميل المشاريع من المصفوفة (Array).
- `assets/`: مجلد يحتوي على الصور المستخدمة في المعرض.
